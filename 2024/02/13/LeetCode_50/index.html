2 Solution: Recursion

Fast Power Algorithm（快速幂算法）是一种用于高效计算幂运算的算法。它利用了指数的二进制表示形式来降低计算复杂度。具体来说，对于指数为 n 的幂运算，传统的方法需要进行 n-1 次乘法操作，而快速幂算法只需要进行 O(log n) 次乘法操作。这种算法通常采用递归或迭代的方式实现，其基本思想是通过将指数 n 分解为二进制形式，然后利用乘法的结合性和交换性质来降低计算的复杂度。这使得快速幂算法在求解指数运算时具有较高的效率和性能。

这段代码是一个计算浮点数 x 的整数次幂的函数，采用了递归和分治的思想。以下是代码的解释和复杂度分析：

  1. 基本情况处理： 如果 n 等于 0，直接返回 1.0，因为任何数的 0 次幂都是 1。

  2. 统一转换为正数次方处理： 为了简化计算，将负数次方的情况转换为正数次方。将 x 变为 1/x，同时将 n 变为 -n。

  3. 判断 n 的奇偶性：

    • 如果 n 是偶数，使用分治思想，将 x 的 n 次幂转化为 x 的平方的 n/2 次幂。递归调用 self.myPow(x ** 2, n >> 1)。
    • 如果 n 是奇数，将 x 的 n 次幂转化为 x 乘以 x 的 n-1 次幂。递归调用 x * self.myPow(x, n - 1)。

复杂度分析：

  • 递归调用会在每一层将问题规模缩小一半（因为 n 右移一位相当于除以 2）。因此，时间复杂度为 O(log n)。
  • 递归调用会使用递归栈，因此空间复杂度也是 O(log n)。这是因为递归栈的深度取决于递归调用的次数，而每次调用时，问题规模都缩小一半。

总体而言，这段代码采用了分治和递归的思想，有效地降低了计算复杂度，使得计算 x 的整数次幂的时间和空间复杂度都是 O(log n)。
